# ArenaIQ Smart Venue Platform

ArenaIQ is a static landing page concept for a smart venue platform that connects fan entry, crowd flow, mobile guidance, concessions, operations, parking, transit, and the underlying intelligence core.

## Run locally

Open `index.html` directly in a browser, or run a local static server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Files

- `index.html` - page structure and content
- `styles.css` - responsive visual design
- `script.js` - sticky header behavior
